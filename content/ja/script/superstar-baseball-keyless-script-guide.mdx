## スタイルシステムの理解

スーパースター・ベースボールにおけるキャラクターの強さの核となるのは「スタイル」です。スタイルは攻撃型と守備型のカテゴリーに分かれており、フィールド上での特殊能力を決定します。2026年にゲームを初めて開始すると、初期ビルドを構築するために4回の無料スピンが付与されます。

### スタイルのレアリティと排出率

キャラクターを最適化したいプレイヤーにとって、確率を知ることは非常に重要です。スタイルのレアリティは、アルティメットパワーアップの威力に直結します。

| レアリティ | 排出率（通常） | 説明 |
| :--- | :--- | :--- |
| **レジェンダリー** | **0.5%** | 最も強力な能力。入手は極めて困難。 |
| **エピック** | **2.0%** | フィールドを大きく支配できる上位ティアの能力。 |
| **レア** | **35.0%** | 信頼できるパワーアップを備えた堅実な中位ティアのスタイル。 |
| **アンコモン** | **62.5%** | 標準的なスタイル。新規プレイヤーに最も一般的。 |

確率を上げるには、**ラッキースピン**を目指すべきです。ラッキースピンは、エピックやレジェンダリーのスタイルを獲得できる確率を大幅に高めます。アクティブなプレイ時間30分ごとに無料スピンを1回獲得できるため、継続性がトップティアのビルドへの鍵となります。

## プレイヤーの役割をマスターする

7対7の対戦では、全プレイヤーがそれぞれの責任を理解する必要があります。タイミングを合わせるために**スーパースター・ベースボール キーレススクリプトガイド**を使用している場合でも、手動のスキルに頼っている場合でも、役割を知ることが勝利への第一歩です。

1.  **バッター：** 主な目的は、ボールをミートして内野を抜くことです。バッターとしての成功は、ピッチャーの球速と変化に合わせてスイングのタイミングを計ることにあります。
2.  **ピッチャー：** 守備の第一線です。目標は、さまざまな球種、速度、変化を駆使してバッターを欺き、三振を奪うか、弱いゴロを打たせることです。
3.  **フィールダー：** フィールダーには高い状況判断能力が求められます。フライをキャッチし、相手チームの得点を防ぐために各ベースへ正確な送球を行うのが仕事です。

## ピッチングの仕組みと球種

ピッチャーの持ち球は最大の武器です。スーパースター・ベースボールでは、1〜4の数字キーに割り当てられた4つの異なる球種を使用できます。ハイレベルなプレイには、これらの球種の「変化（ブレイク）」をマスターすることが不可欠です。

| 球種 | 速度範囲 (MPH) | 変化と特徴 |
| :--- | :--- | :--- |
| **ファストボール** | **80 - 100** | 直球で高速。バッターを力でねじ伏せるために使用。 |
| **スライダー** | **75 - 90** | 鋭く横や下に変化し、目を惑わせる。 |
| **シンカー** | **70 - 90** | ホームプレート付近で急激に沈む。ゴロを打たせるのに最適。 |
| **カーブ** | **65 - 85** | 最も遅い球種で、大きな縦横の変化を伴う。 |

> **💡 プロのヒント：** ファストボールを連発するだけではいけません。カーブやシンカーを使ってバッターの目線を揺さぶり、その後に高めのファストボールを投げて不意を突きましよう。

## FAQ

**Q: 自動ヒットが可能な、機能するスーパースター・ベースボール キーレススクリプトガイドはありますか？**

A: 一部のサードパーティ製スクリプトは自動ヒット機能を謳っていますが、それらはRobloxのアンチチートによって検出されたり、マルウェアを含んでいたりすることがよくあります。最も効果的な「自動ヒット」の方法は、タイミングを練習し、ヒット判定を広げるスタイルを使用することです。

**Q: 新しいコードはどのくらいの頻度でリリースされますか？**

A: コードは通常、ゲームがRobloxで新しい「いいね」のマイルストーン（例：9万いいね、10万いいね）に達したときや、「Subzero」や「Blight」などのシーズンアップデート中にリリースされます。

**Q: プレイヤー間でスタイルをトレードできますか？**

A: 現在、スタイルはキャラクターに紐付けられており、トレードすることはできません。攻撃型と守備型のスタイルをそれぞれ個別にスピンで獲得する必要があります。

**Q: 三振を奪うのに最適な球種は何ですか？**

A: 一般的に**スライダー**が三振を奪うのに最適だとされています。鋭い横の変化により、バッターがストライクゾーン外のボールを振ってしまうことが多いためです。しかし、高速の**ファストボール**と組み合わせるのが最も効果的な戦略です。
