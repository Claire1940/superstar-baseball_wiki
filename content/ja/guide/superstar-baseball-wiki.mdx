## システム要件と準備

インストール作業に入る前に、お使いのハードウェアがProject Rioで使用されるカスタマイズされたDolphinエミュレータを実行できる性能を備えているか確認する必要があります。『ミラクルベースボール』は20年以上前のゲームですが、現代のエミュレーションで投球や打撃のタイミングに不可欠な秒間60フレーム（FPS）を安定して維持するには、それなりのCPU性能が求められます。

### 推奨ハードウェアスペック

| コンポーネント | Windows 最小要件 | Mac 推奨要件 |
|---|---|---|
| **CPU** | Intel i7-7500U または同等品 | Apple Silicon (M1, M2, M3) |
| **GPU** | Nvidia GeForce 940MX | Apple統合GPU |
| **RAM** | 8GB DDR4 | 8GB ユニファイドメモリ |
| **OS** | Windows 10/11 | macOS 12.0 以降 |

PCに加えて、ゲームの合法的なISOファイルが必要です。コミュニティの標準は、修正のない北米版（NTSC-U）の『Mario Superstar Baseball』です。

> **⚠️ 警告：** Project Rioチームおよび本Wikiは、ゲームファイル（ISO）を提供することはできません。これらのファイルを配布することは違法です。ファイルサイズが正確に **1.46 GB** であることを確認してください。これより小さいファイル（nKitや圧縮形式）は、オンラインプレイ中に同期ズレ（デシンク）を引き起こす原因となります。

## WindowsおよびMacへのProject Rioのインストール

Project Rioは、ネットプレイに最適化されたDolphinエミュレータの特殊なフォーク版です。インストール手順はOSによって若干異なります。

### Windowsでのインストール手順
1. 公式サイトからProject Rioの実行ファイルをダウンロードします。
2. Windows Defenderで警告が表示された場合は、「詳細情報」をクリックしてから「実行」を選択してください。
3. インストール先を選択し、ウィザードを完了させます。
4. アプリケーションを起動し、中央の画面をダブルクリックしてISOファイルのあるディレクトリを設定します。

### macOSでのインストール手順
1. Project RioのZIPファイルをダウンロードし、Finderで解凍します（不具合を避けるため、ブラウザのダウンロードメニューから直接開かないでください）。
2. アプリケーションを「アプリケーション」フォルダにドラッグします。
3. 開発元を確認できないというエラーが表示された場合は、**システム設定 > プライバシーとセキュリティ**に移動し、メニュー下部の「このまま開く」を選択してください。
4. プロンプトが表示された場合は、Intelベースのコードとの互換性を確保するためにRosetta 2をインストールしてください。

## コントローラー設定とドライバー

最も本格的な体験を求める多くのプレイヤーは、オリジナルのゲームキューブコントローラーの使用を好みます。ただし、これにはUSBアダプター（Mayflash製や任天堂純正のWii U用アダプターなど）と、入力遅延をなくすための特定のドライバーが必要です。

### ゲームキューブアダプターのセットアップ

| プラットフォーム | 必要なソフトウェア | 重要な設定 |
|---|---|---|
| **Windows** | Zadig | ドライバーを **WinUSB** に変更 |
| **Mac** | GC Adapter Driver | システム設定で「オン」に切り替え |
| **共通** | 物理スイッチ | アダプターを **Wii U モード** に設定 |

PlayStation 5やXbox Series Xのゲームパッドなどの現代的なコントローラーを使用する場合、特別なドライバーは不要です。Project Rioの「Controllers」メニューから「Standard Controller」を選択し、「Configure」をクリックします。入力フィールドをクリックしてからコントローラーの対応するボタンを押すことで、手動でボタンを割り当てる必要があります。

> **💡 ヒント：** XboxやPlayStationのコントローラーをマッピングする際は、プレッシャーのかかる投球場面での「スティックの勝手な動き（ドリフト）」を防ぐため、アナログスティックの「デッドゾーン（Deadzone）」を必ず調整してください。

## オンラインプレイのためのグラフィック最適化

自分と対戦相手の両方がスムーズにプレイできるように、グラフィック設定を最適化する必要があります。同期ズレ（デシンク）の多くは、どちらかのプレイヤーのPCがゲーム速度100%を維持できないことによって発生します。

### 推奨グラフィックバックエンド
*   **macOS：** 常に **Metal** を使用してください。古いガイドではVulkanが推奨されていましたが、現在はMetalがネイティブサポートされており、Appleハードウェアで最高の安定性を提供します。
*   **Windows：** **Direct3D 11** または **Direct3D 12** が通常最も安定しています。古いGPUを使用している場合は、Vulkanの方がわずかにパフォーマンスが向上することがあります。

### パフォーマンス調整表

| 設定項目 | 推奨値 | 効果 |
|---|---|---|
| **Internal Resolution** | 1x または 2x Native | 値を高くすると画質は上がりますが、GPUに負荷がかかります。 |
| **Full Screen** | 「Full Screen」ボタンを使用 | ウィンドウモードよりもFPSの安定性が向上します。 |
| **Show FPS** | 有効 | 60 FPSを下回っていないか監視するのに役立ちます。 |
| **Audio Stretching** | 無効 | ネットプレイ環境でオーディオの遅延を引き起こす可能性があります。 |

## ランク付きコミュニティへの参加

技術的なセットアップが完了したら、**スーパースターベースボールWiki**体験の核心であるオンラインコミュニティへ飛び込みましょう。他のプレイヤーと対戦するには、Project Rioの公式サイトでアカウントを作成する必要があります。このアカウントで、統計、Eloレーティング、トーナメント履歴が記録されます。

### Discordとマッチメイキング
[Mario Baseball Discord](https://discord.gg/mariobaseball)は、すべての試合の中心地です。参加後、以下の手順を踏んでください：
1. **#role-self-assignment** チャンネルに移動します。
2. 「New Netplayer」ロールを選択して、**Pont League**へのアクセス権を取得します。
3. マッチメイキングボットを使用して、「Stars Off（スターなし）」または「Big Balla（スターあり）」モードの待機列に加わります。
4. マッチングしたら、対戦相手から提供されたロビーコードをコピーするか、自分でホストします。

### 競技用ドラフトルール
競技的な『ミラクルベースボール』では、バランスを保つために特定のドラフト形式が採用されています。標準的な「Stars Off」モードでは、コミュニティは **1-1-1-1** のドラフト順序に従います。ただし、クッパやボスパックンといったキャラクターは著しく強力であるため、プレイヤーはしばしば「プレースホルダー（仮の選択）」を使用します。

例えば、ボスパックンをキャプテンにしたいが最初の指名権を持っている場合、まずデイジーなどのプレースホルダーを指名し、ドラフトの後半でボスパックンを選択します。試合開始前に、Project Rioメニューの「Swap Captain（キャプテン交代）」機能を使用して、実際のキャプテンを設定できます。

## 高度な機能：GeckoコードとMod

Project Rioには、Geckoコードをダウンロードできる「Mod」マネージャーが組み込まれています。これらは基本的に、元のゲームコードに機能を追加したりバグを修正したりするコミュニティ製のパッチです。

*   **Training Mode v1.0：** 打撃のタイミングやボールの軌道に関するリアルタイムのフィードバックを提供します。
*   **Unlock All Characters：** チャレンジモードをやり直す必要がないよう、新規インストール時に全キャラクターを解放します。
*   **Custom Stadiums：** マリオサンビーチやワリオパレスなどのクラシックな球場の見た目を変更できるModもあります。

> **⚠️ 警告：** Project Rio内で「チャレンジモード」をプレイしようとしないでください。オンライン専用の修正がメモリーカードの保存に干渉し、クラッシュやデータの破損を招く可能性が高いです。一人用の進行には標準のDolphinを使用してください。

## FAQ

**Q: 高性能なPCを使っているのに、ゲームがラグいのはなぜですか？**

A: オンラインプレイにおけるラグは、多くの場合ハードウェアではなくプレイヤー間の接続が原因です。この**スーパースターベースボールWiki**では、有線LAN接続の使用を強く推奨します。Wi-Fiは本質的に不安定で、スイングのタイミングを狂わせる「カクつき」の原因となります。ピン（Ping）が常に90msを超えている場合、顕著な遅延が発生する可能性があります。

**Q: Steam Deckでオンラインプレイはできますか？**

A: はい！Project RioはDolphinに基づいているため、Steam DeckのようなLinuxベースのシステムでも動作します。デスクトップモードを使用してFlatpak版をインストールするか、特定のコミュニティスクリプトに従ってProject Rioフォークを動作させる必要があります。

**Q: 「Stars On」と「Stars Off」の違いは何ですか？**

A: 「Stars Off」はゲームからスタースキル（必殺技）を排除し、純粋な技術とドラフト戦略に焦点を当てたモードです。「Stars On」（多くの場合Big Balla形式でプレイされる）は、スターポイントを使用して強力な打撃や投球を行うことができ、戦略にリソース管理の要素が加わります。

**Q: Macでゲームキューブコントローラーが認識されません。どうすればいいですか？**

A: システム設定の「ログイン項目と拡張機能」で「GC Adapter Driver」に許可を与えているか確認してください。また、ドライバーを初期化するために、インストール後にPCを再起動する必要があります。物理アダプターが「PC」モードではなく「Wii U」モードに切り替わっていることも常に確認してください。
