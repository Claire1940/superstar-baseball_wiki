## Mécaniques et types de lancers

L'arsenal d'un lanceur est sa plus grande arme. Dans Superstar Baseball, vous disposez de quatre types de lancers distincts, chacun mappé sur les touches numériques 1 à 4. Maîtriser l'effet (break) de ces lancers est essentiel pour le jeu de haut niveau.

| Type de lancer | Plage de vitesse (MPH) | Mouvement et caractéristiques |
| :--- | :--- | :--- |
| **Balle rapide** | **80 - 100** | Droite et à haute vélocité ; utilisée pour surprendre les batteurs. |
| **Glissante (Slider)** | **75 - 90** | Cassure latérale et descendante nette ; trompe l'œil. |
| **Plongeante (Sinker)** | **70 - 90** | Tombe soudainement au niveau de la plaque ; idéal pour les balles roulantes. |
| **Balle courbe** | **65 - 85** | Lancer le plus lent avec une forte courbe descendante/latérale. |

> **💡 Conseil de Pro :** Ne vous contentez pas d'enchaîner les balles rapides. Utilisez la balle courbe ou la plongeante pour modifier le niveau de vision du batteur, puis enchaînez avec une balle rapide haute pour le prendre au dépourvu.
